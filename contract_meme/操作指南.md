设置sepolia网段地址   
    输入指令：npx hardhat keystore set SEPOLIA_RPC_URL
    输入SEPOLIA_RPC_URL的值

设置钱包的私钥  
    输入指令：npx hardhat keystore set SEPOLIA_PRIVATE_KEY
    输入SEPOLIA_PRIVATE_KEY的值

部署合约到sepolia网上
    npx hardhat ignition deploy ./ignition/modules/MeMeToken.ts --network sepolia
